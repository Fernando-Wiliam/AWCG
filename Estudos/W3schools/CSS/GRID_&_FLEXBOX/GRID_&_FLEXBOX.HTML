DICAS SOBRE FLEXBOX E GRID

  • Flexbox é unidimensional, ou seja, linha OU coluna.
    Perfeito para COMPONENTES

  • CSS Grid é multidimencional, ou seja, linhas E colunas.
    Perfeito para LAYOUTS

Então podemos criar Componentes com Flexbox e usar dentro de layots em CSS Grid
